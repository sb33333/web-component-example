사용자 정의 요소. SHADOW DOM에 정의한 STYLE + STYLESHEET 적용

SHADOW DOM 스타일 > STYLESHEET 스타일

참고

shadow DOM 내부의 input 요소는 일반적으로 form에서 인식되지 않지만 formAssociated 속성을 사용해 form에서 인식할 수 있습니다.

shadowRoot.adoptedStyleSheets를 사용해 외부의 css style sheet를 적용할 수 있습니다.
style 적용 우선운위는 shadow DOM에 선언된 style 요소가 먼저 적용되고 불러온 외부 style sheet가 적용됩니다.
