# web-component
## Web Component?
### 재사용할 수 있는 custom elements를 만들 수 있는 기술.
1. Custom elements
  - 사용자 정의 요소와 그 동작을 정의할 수 있는 JavaScript API.
2. Shadow DOM
  - Element에 캡슐화된 DOM 트리를 생성하기 위한 JavaScript API.
3. HTML template element
  - 마크업을 작성하고 재사용하기 위한 <template>과 <slot>요소.
